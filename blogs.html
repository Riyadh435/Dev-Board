Question-1: What are the different ways to select an element in the DOM?

Answer : 1. getElementById(): This method is used to select an element by its id attribute. 2. getElementsByClassName(): This method is used to select elements by their class name. 3. getElementsByTagName(): This method is used to select elements by their tag name. 4. querySelector(): This method is used to select the first matching element that matches a specified CSS selector. 5. querySelectorAll(): This method is used to select all elements that match a specified CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : innerHTML: The innerHTML property sets or returns the HTML content (inner HTML) of an element. innerText: The innerText property sets or returns the text content of the specified node, and all its descendants. textContent: The textContent property sets or returns the text content of the specified node, and all its descendants.

Question-3: What is event delegation in the DOM?

Answer : Event delegation is a technique in which we attach a single event listener to a parent element that will fire for all descendants matching a selector, whether they are present now or added in the future.

Question-4: What is event bubbling in the DOM?

Answer : Event bubbling is a type of event where the event first triggers on the innermost target element, and then successively triggers on the parents of the target element.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer : To create an element, we can use the document.createElement() method. To add an element to the DOM, you can use the appendChild() method. To remove an element from the DOM, we can use the removeChild() method.
